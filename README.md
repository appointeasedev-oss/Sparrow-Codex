# Sparrow AI

Sparrow AI is an open-source AI-powered web development assistant created by Sparrow AI Solutions. It allows users to generate complete web applications, including HTML, CSS, and JavaScript, by providing natural language prompts. The application leverages the power of various large language models through OpenRouter to generate code and provide a seamless development experience.

## Features

*   **AI-Powered Code Generation**: Describe the website or component you want to build, and Sparrow AI will generate the code for you.
*   **Live Preview**: See a real-time preview of your generated application as you make changes.
*   **Image-to-Code**: Provide an image of a website, and Sparrow AI will analyze it and generate the corresponding code.
*   **Wide Range of Models**: Choose from a variety of large language models available through OpenRouter, including free and paid options.
*   **Local API Key Storage**: Your OpenRouter API key is stored securely in your browser's local storage and is never transmitted to our servers.

## Getting Started

To get started with Sparrow AI, follow these simple steps:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/your-repository/sparrow-ai.git
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    # or
    pnpm install
    ```

3.  **Run the Development Server**:
    ```bash
    npm run dev
    # or
    pnpm dev
    ```

4.  **Open in Browser**:
    Open your browser and navigate to `http://localhost:3000`. You will be prompted to enter your OpenRouter API key to begin.
